# 📊 Dashboard Service Desk - Power BI

## 🎯 Objetivo
Analisar indicadores operacionais de Service Desk com foco em SLA, TMA, Aging e volumetria de chamados.

---

## 📈 Principais Indicadores
- SLA (% dentro e fora do prazo)
- Tempo Médio de Atendimento (TMA)
- Aging dos chamados
- Volume por analista
- Ranking de ofensores (categorias)

---

## 🛠 Ferramentas utilizadas
- Power BI
- Excel
- DAX
